# Small Python Projects

Just a collection of small projects I built while learning Python.

## Projects

### Wordle CLI
A terminal clone of Wordle. Gives color-coded feedback on your guesses — green for right spot, yellow for wrong spot, red for not in the word at all. Built with the `rich` library.

```bash
pip install rich
python wordle.py
```
